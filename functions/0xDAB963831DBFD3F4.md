# WEAPON::IS_AIR_DEFENCE_SPHERE_IN_AREA

## Summary
Returns true if an air defence sphere is detected in the specified zone. Returns the found sphere index by reference.

## Parameters
* **FLOAT** positionX
* **FLOAT** positionY
* **FLOAT** positionZ
* **FLOAT** radius
* **INT\*** sphereIndex

## Returns
* **BOOL**
