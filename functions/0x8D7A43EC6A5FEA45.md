# MISC::GET_COORDS_OF_PROJECTILE_TYPE_IN_AREA

## Summary
Checks that a projectile object of the specified type is in the area and returns it's position

## Parameters
* **FLOAT** minCoorsX
* **FLOAT** minCoorsY
* **FLOAT** minCoorsZ
* **FLOAT** maxCoorsX
* **FLOAT** maxCoorsY
* **FLOAT** maxCoorsZ
* **WEAPON_TYPE** weaponType
* **VECTOR\*** positionOut
* **BOOL** isPlayer = false

## Returns
* **BOOL**
