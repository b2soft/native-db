# STATS::PLAYSTATS_MC_CLUBHOUSE_ACTIVITY

## Summary
Telemetry sent whenever any of the actvity events finishes (Darts, Arm Wrestling, Drinking, Smoking, Jukebox)
If not a jukebox activity, jukeboxStation should be set to -1

## Parameters
* **INT** bossId1
* **INT** bossId2
* **INT** matchId1
* **INT** matchId2
* **INT** minigameType
* **INT** jukeboxStation
* **INT** mcPointsEarned
