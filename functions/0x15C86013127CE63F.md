# TASK::TASK_BOAT_MISSION

## Summary
Gives the boat a mission

## Parameters
* **PED_INDEX** pedIndex
* **VEHICLE_INDEX** boatIndex
* **VEHICLE_INDEX** targetVehicleIndex: (If a target vehicle needs to be specified it is done here)
* **PED_INDEX** targetPedIndex: (If a target ped needs to be specified it is done here) FLOAT TargetX, TargetY, TargetZ (specify targetcoordinates here in needed.)
* **FLOAT** coorsX
* **FLOAT** coorsY
* **FLOAT** coorsZ
* **VEHICLE_MISSION** mission
* **FLOAT** cruiseSpeed: (in m/s)
* **DRIVINGMODE** mode
* **FLOAT** targetReachedDist: (distance (in meters) at which plane thinks it's arrived.) ALSO used as the hover distance for MISSION_ATTACK and MISSION_CIRCLE
* **BOATMODE** boatFlags

## Description
<!-- AUTOGENERATED: Remove this line before editing -->
You need to call PED::SET_BLOCKING_OF_NON_TEMPORARY_EVENTS after TASK_BOAT_MISSION in order for the task to execute.

Working example
float vehicleMaxSpeed = VEHICLE::GET_VEHICLE_ESTIMATED_MAX_SPEED(ENTITY::GET_ENTITY_MODEL(pedVehicle));
TASK::TASK_BOAT_MISSION(pedDriver, pedVehicle, 0, 0, waypointCoord.x, waypointCoord.y, waypointCoord.z, 4, vehicleMaxSpeed, 786469, -1.0, 7);
PED::SET_BLOCKING_OF_NON_TEMPORARY_EVENTS(pedDriver, 1);
