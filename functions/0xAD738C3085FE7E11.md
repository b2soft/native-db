# ENTITY::SET_ENTITY_AS_MISSION_ENTITY

## Summary
Sets a random entity as a mission entity

## Parameters
* **ENTITY_INDEX** entityIndex
* **BOOL** scriptHostObject = true:
If true, this entity has been created by the host portion of a network script and is vital to that script - it must always exist regardless of who is hosting the script.
If false, the entity has been created by the client portion of a network script and can be removed when the client who created it leaves the script session.
* **BOOL** grabFromOtherScript = false: If true, this entity will be grabbed off any script that currently owns it
