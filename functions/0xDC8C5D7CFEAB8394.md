# MISC::HAS_BULLET_IMPACTED_IN_BOX

## Summary
Finds a bullet impact in the box

## Parameters
* **FLOAT** minCoorsX
* **FLOAT** minCoorsY
* **FLOAT** minCoorsZ
* **FLOAT** maxCoorsX
* **FLOAT** maxCoorsY
* **FLOAT** maxCoorsZ
* **BOOL** isPlayer = true: Checks if the player fired the bullet.
* **BOOL** entryOnly = true:
TRUE will only find entry impacts.
 =FALSE will find both entry and exit impacts

## Returns
* **BOOL**
