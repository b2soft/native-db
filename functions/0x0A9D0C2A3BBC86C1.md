# TASK::DOES_SCENARIO_OF_TYPE_EXIST_IN_AREA

## Summary
Returns true if there is a scenario of the specific type in the area given

## Parameters
* **FLOAT** positionX
* **FLOAT** positionY
* **FLOAT** positionZ
* **STRING** scenarioType
* **FLOAT** range
* **BOOL** mustBeFree: Is true it will only return true if the point is not in use.

## Returns
* **BOOL**
