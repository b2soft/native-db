# VEHICLE::START_PLAYBACK_RECORDED_VEHICLE_WITH_FLAGS

## Summary
Starts the playback for a recorded vehicle

## Parameters
* **VEHICLE_INDEX** vehicleIndex
* **INT** fileNumber
* **STRING** recordingName
* **INT** flags = 4: Can be from the enum VEHICLE_RECORDING_PLAYBACK_FLAGS, then can be combined together using "|"
* **INT** delayInMillisecondsToRevertIfSwitchedToAI = 0:
Is used to specify how long the vehicle will wait before reverting back to a normal recording.
IF 0 is specified (the default) the car will never revert back
* **DRIVINGMODE** mode = 786603
