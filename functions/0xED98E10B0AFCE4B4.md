# TASK::SET_PED_WAYPOINT_ROUTE_OFFSET

## Summary
Sets a worldpsace offset on the ped's current TASK_FOLLOW_WAYPOINT_RECORDING, which is applied to all the route points for this ped only

## Parameters
* **PED_INDEX** ped
* **FLOAT** offsetX
* **FLOAT** offsetY
* **FLOAT** offsetZ

## Returns
* **BOOL**
