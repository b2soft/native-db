# ENTITY::CREATE_MODEL_HIDE_EXCLUDING_SCRIPT_OBJECTS

## Summary
This commands does the same as CREATE_MODEL_HIDE except that it won't hide objects owned by script

## Parameters
* **FLOAT** posX
* **FLOAT** posY
* **FLOAT** posZ
* **FLOAT** radius
* **MODEL_NAMES** modelHash
* **BOOL** surviveMapReload
