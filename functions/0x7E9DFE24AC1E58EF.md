# PED::IS_PED_SHOOTING_IN_AREA

## Summary
Checks that a ped is shooting in an area

## Parameters
* **PED_INDEX** pedIndex
* **FLOAT** minX
* **FLOAT** minY
* **FLOAT** minZ
* **FLOAT** maxX
* **FLOAT** maxY
* **FLOAT** maxZ
* **BOOL** highlightArea
* **BOOL** do3DCheck = true

## Returns
* **BOOL**
