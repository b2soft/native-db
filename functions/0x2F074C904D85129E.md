# PED::SET_COP_PERCEPTION_OVERRIDES

## Summary
Allows script to override the cop perception parameters. This affects all code-spawned, non-mission cops. Parameters will not be overridden if left at their default values (these are the standard values already used by cops).

## Parameters
* **FLOAT** seeingRange = 60:
How far away a ped can identify the ped (default on foot: 60.0f, in car/boat: 90.0f, heli: 150.0f).
Peripheral: how far the perhiperal vision extends (doesn't affect rendered cone blip).
* **FLOAT** seeingRangePeripheral = 5
* **FLOAT** hearingRange = 60: How far a ped can hear (default on foot: 60.0f, in car/boat: 90.0f, heli: 150.0f) (doesn't affect rendered cone blip).
* **FLOAT** minAzimuthAngle = -90: MIN horizontal field of view for the ped (doesn't affect rendered cone blip).
* **FLOAT** maxAzimuthAngle = 90: MAX horizontal field of view for the ped (doesn't affect rendered cone blip).
* **FLOAT** centreOfGazeMaxAngle = 60: Defines the central angle.
* **FLOAT** rearViewSeeingRangeOverride = -1:
Override rear view mirror perception range (when in vehicle).
Will use default range (fSeeingRange) if set to -1.0f.
More info on all of these params in the wiki.
