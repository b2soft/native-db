# STATS::PLAYSTATS_MC_FORMATION_ENDS

## Summary
Telemetry sent when formation ENDS
formationStyle - formation Style
duration - how long formation lasted on the road
membersAmount - Max number of MC members involved

## Parameters
* **INT** bossId1
* **INT** bossId2
* **INT** matchId1
* **INT** matchId2
* **INT** formationStyle
* **INT** duration
* **INT** membersAmount
