# OBJECT::IS_PLAYER_ENTIRELY_INSIDE_GARAGE

## Summary
Returns TRUE if the player is entirely inside the garage.  If the player is in a vehicle it uses the vehciles bound volume.

## Parameters
* **INT** garageHash
* **PLAYER_INDEX** playerIndex
* **FLOAT** margin = 0
* **INT** boxIndex = -1

## Returns
* **BOOL**
