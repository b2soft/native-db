# NETWORK::NETWORK_SEND_INVITE_VIA_PRESENCE

## Summary
Send session invite via presence
Pass NULL to content ID to indicate freemode invite
Pass 0 to nPlaylistLength to indicate no playlist

## Parameters
* **GAMER_HANDLE\*** gamer
* **STRING** szContentID
* **INT** playlistLength
* **INT** playlistCurrent

## Returns
* **BOOL**
