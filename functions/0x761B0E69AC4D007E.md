# OBJECT::HAS_CLOSEST_OBJECT_OF_TYPE_BEEN_BROKEN

## Summary
Does a world search and returns TRUE if a broken object with the specified model is found

## Parameters
* **FLOAT** scrVecCoorsX
* **FLOAT** scrVecCoorsY
* **FLOAT** scrVecCoorsZ
* **FLOAT** radius
* **MODEL_NAMES** model
* **SEARCH_LOCATION_FLAGS** searchFlags

## Returns
* **BOOL**
