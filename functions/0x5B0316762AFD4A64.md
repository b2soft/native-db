# GRAPHICS::GET_STATUS_OF_CREATE_MISSION_CREATOR_PHOTO_PREVIEW

Once this preview is ready, it can be used by script or scaleform. Both the name of the TXD and the texture is always
"MISSION_CREATOR_TEXTURE"
