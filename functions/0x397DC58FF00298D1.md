# OBJECT::IS_ANY_OBJECT_NEAR_POINT

## Summary
Returns true if any object's pivot point lies within the sphere with the given radius at the given point

## Parameters
* **FLOAT** pointX
* **FLOAT** pointY
* **FLOAT** pointZ
* **FLOAT** radius
* **BOOL** considerScriptCreatedObjectsOnly = false

## Returns
* **BOOL**
