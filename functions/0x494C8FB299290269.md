# NETWORK::NETWORK_ADD_ENTITY_AREA

## Summary
Adds an entity area with the specified start and end points
The return value is a script ID that can be used to refer to the same entity area again

## Parameters
* **FLOAT** startX
* **FLOAT** startY
* **FLOAT** startZ
* **FLOAT** endX
* **FLOAT** endY
* **FLOAT** endZ

## Returns
* **INT**
