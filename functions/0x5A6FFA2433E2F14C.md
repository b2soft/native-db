# NETWORK::NETWORK_START_RESPAWN_SEARCH_FOR_PLAYER

## Summary
This command initiates a search for spawn coordinates using the navmesh, using a spherical volume

## Parameters
* **PLAYER_INDEX** playerIndex: The index of the player to respawn vSearchPos - the location where you wish to find a spawn coordinate
* **FLOAT** searchPosX
* **FLOAT** searchPosY
* **FLOAT** searchPosZ
* **FLOAT** searchRadius: The range of the search vFaceTowardsPos - the position which you wish the spawn coordinate to face towards
* **FLOAT** faceTowardsPosX
* **FLOAT** faceTowardsPosY
* **FLOAT** faceTowardsPosZ
* **RESPAWN_QUERY_FLAGS** flags = 0: Optional flags

## Returns
* **BOOL**
