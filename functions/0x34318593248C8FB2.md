# MISC::IS_PROJECTILE_TYPE_WITHIN_DISTANCE

## Summary
Checks that a projectile object of the specified type is in a sphere

## Parameters
* **FLOAT** vecCoorsX
* **FLOAT** vecCoorsY
* **FLOAT** vecCoorsZ
* **WEAPON_TYPE** weaponType
* **FLOAT** distance
* **BOOL** isPlayer = false

## Returns
* **BOOL**
