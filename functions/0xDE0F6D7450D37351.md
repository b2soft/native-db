# MISC::IS_BULLET_IN_BOX

## Summary
Checks that a bullet is in the area

## Parameters
* **FLOAT** minCoorsX
* **FLOAT** minCoorsY
* **FLOAT** minCoorsZ
* **FLOAT** maxCoorsX
* **FLOAT** maxCoorsY
* **FLOAT** maxCoorsZ
* **BOOL** isPlayer = true: Checks if the player fired the bullet.

## Returns
* **BOOL**
